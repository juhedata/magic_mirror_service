# Magic_Mirror


## 前言

`Magic_Mirror`项目致力于打造基于视频流无感知考勤系统，采用现阶段流行技术实现。


## 项目介绍

`Magic_Mirror`项目是一套基于视频流无感知考勤系统，包括后端管理系统、统计大屏、视频流考勤界面，基于SpringBoot+MyBatis实现。
后台管理系统包含人员管理、监控分析、用户管理、权限管理、设置等模块。
